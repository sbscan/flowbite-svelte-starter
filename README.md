# Flowbite-Svelte-Starter

## Feature

- Svelte
- SvelteKit
- Tailwind CSS
- Flowbite
- Flowbite-Svelte
- ESlint
- Typescript
- Playwright
- Prettier
- Svelte-heros(Heroicons)
- Darkmode activated

## Installation



```sh
npx degit shinokada/flowbite-svelte-starter my-demo
cd my-demo
pnpm i // or npm i
pnpm run dev // or npm run dev
```
